## ⚡ Why This Project?

Understanding how code executes is one of the biggest challenges for beginners.

This tool simplifies learning by:

* Converting code into **visual flowcharts**
* Explaining logic in **simple, natural language**
* Showing execution flow **step-by-step**

---

## 🚀 Features

* 🧠 **AI-Powered Flowcharts**
  Generate logical flowcharts from code using Google Gemini API

* 💻 **Interactive Code Editor**
  Write and test code in Java, Python, JavaScript, C, and C++

* 📖 **Natural Language Explanations**
  Beginner-friendly explanations of code logic

* 🔄 **Execution Flow Visualization**
  Understand how code runs step-by-step

* 🔐 **Authentication System**
  Secure login using AWS Cognito or continue as Guest

* 🕓 **Session History**
  Save and revisit previous debugging sessions

---

## 🧠 Architecture

* **Frontend:** React.js
* **Backend:** Node.js + Express
* **AI Integration:** Google Gemini API
* **Authentication:** AWS Cognito
* **Deployment:** Previously deployed on AWS

## ⚡ Why This Project?

Understanding code execution is one of the biggest challenges for beginners.
Most tools explain *what* code does, but not *how* it runs internally.

This project bridges that gap by:

* Converting code into **visual flowcharts**
* Explaining logic in **simple, natural language**
* Showing execution flow **step-by-step**

---

## 🚀 Features

* 🧠 **AI-generated flowcharts from code**
* 💻 **Multi-language support** (Java, Python, JavaScript, C, C++)
* 📖 **Beginner-friendly code explanations**
* 🔄 **Step-by-step execution visualization**
* 🔐 **Secure authentication** using AWS Cognito
* 🕓 **Session history tracking**

---

## 📈 Impact

* Helps beginners visualize and understand code execution
* Simplifies complex logic using AI-generated explanations
* Bridges the gap between theory and practical understanding

---

## 🧠 Architecture

* **Frontend:** React.js
* **Backend:** Node.js + Express
* **AI Integration:** Google Gemini API
* **Authentication:** AWS Cognito
* **Deployment:** Previously deployed on AWS
